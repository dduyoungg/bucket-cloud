# Bucket Cloud

화이트톤의 모던/키치 버킷리스트 캔버스 UI 프로토타입입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 아래 주소를 열면 됩니다.

```text
http://localhost:3000
```

## 현재 포함된 기능

- 섹션 추가
- 섹션별 필터
- 버킷 추가
- 버킷 글씨 드래그 이동
- 글씨 색상 / 크기 / 굵기 / 폰트 변경
- 완료 처리 시 아래 영역으로 가라앉아 쌓이는 구조
- 취소선 없이 완료 버킷 보관
- localStorage 저장
